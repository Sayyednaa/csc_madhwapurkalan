# CSC Centre Website - Complete Project Prompt

Build a modern, premium, responsive Django 5 web application for my CSC (Common Service Centre). The website should primarily function as a business landing page with a custom admin dashboard for managing customer inquiries and website content.

## Tech Stack

* Django 5 (Latest)
* Django Templates
* HTMX
* Alpine.js
* Bootstrap 5 (or Tailwind if it results in a cleaner UI)
* SQLite (default, easily switchable to PostgreSQL)
* Mobile-first responsive design
* Clean component-based structure

## Design Theme

The website should feel modern, minimal, trustworthy, and premium.

Use:

* Soft shadows
* Rounded cards
* Smooth animations
* Gradient hero section
* Professional icons
* Fast loading
* Excellent mobile experience
* SEO-friendly structure

Do NOT use the default Django Admin for day-to-day operations.

Create a beautiful custom admin panel.

Dashboard should display:

* Total Inquiries
* Today's Inquiries
* Appointments
* Callback Requests
* Unread Messages

Use charts where appropriate.
